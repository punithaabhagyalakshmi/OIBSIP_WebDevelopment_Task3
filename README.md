# OIBSIP_WebDevelopment_Task3
A simple dark-themed Temperature Converter built using **HTML, CSS, and JavaScript**.  
It converts temperatures between **Celsius and Fahrenheit** with instant results and input validation.

# Features

- Convert Celsius → Fahrenheit  
- Convert Fahrenheit → Celsius  
- Input validation for non-numeric values  
- Dark theme with glowing UI  
- Clean and responsive layout  

# Technologies Used

- HTML5  
- CSS3  
- JavaScript (DOM Manipulation & Events)
